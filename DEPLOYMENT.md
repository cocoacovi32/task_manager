## How to Deploy Updates

1. Make changes to `/frontend` folder
2. Commit: `git add . && git commit -m "Your message"`
3. Push: `git push`
4. Netlify auto-deploys (watch https://app.netlify.com)
5. Site updates in 2-3 minutes
